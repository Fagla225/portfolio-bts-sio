Pourquoi le responsive ?

Aujourd'hui, plus de la moitié des personnes naviguent sur internet depuis leur téléphone. Un site qui ne s'adapte pas aux petits écrans est difficile à lire et perd des visiteurs.

Le responsive design permet d'avoir un seul code HTML/CSS qui fonctionne bien sur tous les appareils.

Technologies utilisées

Balises HTML

Le HTML structure le contenu de la page avec des balises comme h1, p, section, nav...

CSS

Le CSS donne du style à la page : couleurs, tailles, espacements, polices...

Media Queries

Les media queries permettent d'appliquer un style différent selon la taille de l'écran.
